# Health Wallet

Decentralize App is a simple Medical Record' App to practice the creation of a blockchain applicaciont using Solidity and Javascript.

![](./screenshot.png)

The stack and tools of this application is:

* Truffle Framework
* Ganache
* Solidity
* Nodejs
* Bootstrap# zencon2
* Vuejs

Para correr

```
truffle migrate
```

```
npm run dev
```
